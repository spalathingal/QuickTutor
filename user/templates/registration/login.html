About QuickTutor

QuickTutor is a web application designed to connect students to nearby tutors.

As a tutee, you will be able to:

  • Make a tutee profile to send requests for help
  • Access a list of available tutors
  • Send requests to tutors for help

As a tutor, you will be able to:

  • Make a tutor profile showing subjects you can help in
  • Get requests for help based on your profile
  • Receive payment for your services
